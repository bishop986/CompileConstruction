### C Minus 词法规则

1. 关键字: if, else, int, return, void, while

2. 专用符号: + - * / > >= <= > == != = . ( ) [ ] { } /* */ ;

3. 正规表达式:
	- ID = [a-zA-Z]+
	- NUM = [0-9]+

4. 空白符包括空格，换行，制表符

5. 注释由/*...*/围起来
